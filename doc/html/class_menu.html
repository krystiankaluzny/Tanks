Opis szczegółowy

Klasa wyboru trybu gry: 1 gracz czy 2 graczy lub wyjścia. Klasa jest pierwszym stanem aplikacji, pojawia się zaraz po uruchomieniu programu i pozwala na przejście do stanu gry (klasa Game).

Implementuje AppState.

void Menu::eventProcess ( SDL_Event *  ev)
virtual

Funkcja odpowiada na reakcję na klawisze:

  • Strzałka w górę i w dół - zmiana wybranej pozycji menu
  • Enter i Spacja - zatwierdzenie obecnej pozycji menu
  • Esc - wyjście z programu
    Parametry
    ev- wskaźnik na unię SDL_Event przechowującą typ i parametry różnych zdarzeń

bool Menu::finished ( ) const
virtual

Funkcja sprawdzająca czy należy zakończyć stan menu i przejści do kolejnego stanu gry.

Zwraca
true jeżeli została wybrana, któraś z opcji menu lub został wciśnięty kalwisz Esc, false w przeciwnym wypadku
