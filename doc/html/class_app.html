Funkcja obsługująca zdarzenia czekające w kolejce.

void App::run ( )

Funkcja zawiera inicjalizację biblioteki SDL2, silnika gry, wczytanie tekstur oraz czcionek. Po poprawnej inicjalizacji następuje wejście w pętlę główną programu, która kolejno: reaguje na zdarzenie, uaktualnia obecny stan aplikacji, rysuje obiety na ekranie.
